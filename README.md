# 🌡️ Advanced Temperature Converter

A modern and responsive **Temperature Converter Web App** built using **HTML, CSS, and JavaScript**.  
It allows users to convert temperature between **Celsius, Fahrenheit, and Kelvin** with a clean UI and advanced features.

---

## 🚀 Live Demo
👉 https://shivank302.github.io/temperature-converter-/

---

## ✨ Features

- 🔄 Convert between **Celsius, Fahrenheit, and Kelvin**
- 🔁 **From → To selection** (any unit to any unit)
- 🌙 **Dark Mode Toggle**
- 📜 **Conversion History**
- ⚡ Instant results with smooth UI
- 🎨 Clean and responsive design (mobile-first)
- ⌨️ Enter key support for quick conversion

---

## 🛠️ Tech Stack

- **HTML5**
- **CSS3**
- **JavaScript (Vanilla JS)**
